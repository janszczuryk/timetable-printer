# Timetable printer

## Prerequisites
- Python 3.11

## Install
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt 
```

## Run

Classic approach:
```bash
python3 ./printer.py
```

JetBrains IDE approach:

Click *run button* (Shift+F10) when script *printer* is selected.
